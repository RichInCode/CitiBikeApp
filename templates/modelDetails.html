Here are some details about the model used for predictions:

The trained model is a Random Forest Regressor Model. The Random Forest is trained with 20 estimators. This has been evaluated by doing a grid search.

The importance of each feature has been evaluated and displayed in the graph at left.

The main drivers in the number of rides seems to be neighborhood followed by temperature followed by weekend travel.

The model has been trained and through cross-validation is found to have a score of 0.94.
